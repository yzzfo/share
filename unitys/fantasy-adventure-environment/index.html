      • Fantasy Adventure Environment

      An environment asset pack inspired by the current 3D adventure games and of old. This package focuses on providing artistically crafted environment assets.
      文档 | PC 演示 | 论坛主题

      最高可兼容 Unity 2020.1（LWRP 或 HDRP 例外）。

      最新推出！Unity 2019.3+ 现在支持通用渲染管线。修订了水着色器，以进行完善。可通过风格化草着色器单独获取 URP 草着色器

      受流行 3D 冒险游戏和旧版本启发的环境资源包。该资源包侧重于提供艺术化制作的环境资源。

      当您的英雄角色穿越世界去冒险时，景观中点缀着风格化的植被！

      == 兼容性 ==
      所有资源的设计都针对中高端 PC 和控制台平台 (PS4/Xbox One)。根据 1750 多份基准报告显示，可下载的演示平均运行帧率为 98 FPS。

      不支持移动端和 Web 平台。PC 上的 VR 已通过测试，没有任何问题。完整细节。

      == 功能 ==
      ➮用于树木和植被的自定义着色器，可以启用风格化的渲染和动画
      ➮ 树木和植被上逼真的风动画
      ➮ 植被从玩家身旁弯曲掠过（URP 中不可用）
      ➮ 草色与地形浑然一体（支持倾斜地形和网格地形，URP 中不可用）
      ➮ 多细节层次和碰撞网格适用于所有需要它们的资源
      ➮ 广告牌树多细节层次，可在开放的景观中实现最佳性能
      ➮ 量身定制的环境场景示例

      == 内容 ==
      ▪ 18 种植物
      ▪ 3 种云杉
      ▪ 3 种桦树
      ▪ 3 种风格化的树
      ▪ 2 种柳树
      ▪ 3 种棕榈树
      ▪ 1 种松树
      ▪ 6 种树皮材质
      ▪ 6 种地形材质
      ▪ 7 种悬崖，以增加您的地形（2k 纹理）
      ▪ 4 种小石块
      ▪ 匹配的 skybox 全景（最高 8k 分辨率）
      ▪ 镜头光晕纹理图集
      ▪ 动画化的局部阳光照射特效
      ▪ 雾片特效
      ▪ 落叶粒子效果
      ▪ 风痕粒子效果
      ▪ 后处理栈/ UPP 配置文件和颜色分级纹理（适用于 v1 和 v2）
      ▪ 旧版风格化水着色器（URP 安装特有一个重新设计的版本）

      == 其他功能 ==
      ▪ 着色器支持自定义 SpeedTree 模型
      ▪ 支持 GPU（间接）实例化
      ▪ Vegetation Studio Pro 集成的着色器控制器
      ▪ 具有（三向）全局彩色贴图和细节法线的悬崖着色器
      ▪ 悬崖覆盖物（雪、草、沙等）
      ▪ 树着色器支持颜色变化
      ▪ 具有文档切换功能的着色器 GUI
      ▪ 可编辑的 Amplify Shader Editor​​​ 或 Shader Graph 着色器

      ▪ Gaia
